Компонент AlfaPribor.VideoCameras3

версия компонента для работы с телекамерами:
- Axis, 
- видеосервер Axis, 
- Basler, 
- Beward, 
- видеосервер Beward, 
- HikVision
- возможна доработка для получения видеопотока с любой камеры, 
  поддерживающей передачу видеопотока MJPEG по протоколам HTTP или RTSP
